# 朗读功能使用指南

本网站为每篇新闻都配备了智能朗读功能，让您可以通过语音收听新闻内容。

## 🎯 功能概览

朗读工具栏位于每个新闻页面的顶部，提供以下功能：

### 主要功能

1. **开始朗读** 🔊
   - 点击"朗读新闻"按钮即可开始
   - 系统会自动提取页面文本内容
   - 按句子分段朗读，更自然流畅

2. **暂停/继续** ⏸️ ▶️
   - 朗读时可随时暂停
   - 再次点击继续上次的位置
   - 不会丢失进度

3. **停止朗读** ⏹️
   - 完全停止朗读
   - 重置所有进度
   - 可重新开始

4. **调节速度** ⚡
   - 使用滑块调整朗读速度
   - 范围：0.5x（慢速）至 2.0x（快速）
   - 默认速度：1.0x（正常）
   - 实时生效，无需重启

5. **进度显示** 📊
   - 进度条显示朗读完成百分比
   - 实时显示当前朗读的文本
   - 便于跟踪朗读位置

## 📝 使用步骤

### 第一步：打开新闻页面
从首页点击任意新闻分类卡片，进入具体的新闻页面。

### 第二步：找到朗读工具栏
在页面顶部（标题下方）会看到朗读控制栏，包含：
- 朗读按钮
- 速度控制滑块
- 进度显示区域

### 第三步：开始朗读
点击"🔊 朗读新闻"按钮，系统会：
1. 提取页面文本内容
2. 智能分割为句子
3. 开始逐句朗读

### 第四步：调整体验（可选）
根据个人喜好调整：
- **慢速朗读**（0.5x - 0.9x）：适合学习、记笔记
- **正常速度**（1.0x）：日常收听
- **快速朗读**（1.1x - 2.0x）：快速浏览、节省时间

### 第五步：控制朗读
使用控制按钮：
- 需要暂停时点击 ⏸️ 按钮
- 想继续时点击 ▶️ 按钮
- 要停止时点击 ⏹️ 按钮

## 💡 使用技巧

### 推荐速度设置
- **详细阅读**：0.8x - 1.0x
- **日常浏览**：1.0x - 1.2x
- **快速浏览**：1.3x - 1.5x
- **极速浏览**：1.5x - 2.0x

### 最佳使用场景
- 📱 通勤路上收听新闻
- 💻 工作时多任务处理
- 👀 眼睛疲劳时休息
- 🎧 配合耳机获得更好体验
- 📚 学习新知识时加深记忆

### 注意事项
1. 首次使用时浏览器可能请求权限，请允许
2. 确保设备音量已打开
3. 朗读时切换页面会自动停止
4. 某些特殊符号可能影响朗读效果
5. 建议使用现代浏览器（Chrome、Edge、Safari）

## 🔧 技术特性

- **零依赖**：使用浏览器原生 Web Speech API
- **离线可用**：无需网络连接
- **隐私保护**：数据不会上传到服务器
- **性能优化**：智能文本分割，流畅不卡顿
- **自适应**：自动识别中文内容

## 🐛 常见问题

### Q: 为什么听不到声音？
A: 请检查：
1. 设备音量是否打开
2. 浏览器是否静音
3. 是否在浏览器中允许了音频权限

### Q: 朗读速度可以更快吗？
A: 最高支持 2.0x 速度，这是浏览器 API 的限制。

### Q: 可以选择朗读部分内容吗？
A: 当前版本朗读完整页面内容，后续版本会考虑添加选择性朗读。

### Q: 支持其他语言吗？
A: 当前优化了中文朗读，其他语言取决于浏览器支持。

### Q: 朗读时可以做其他事吗？
A: 可以！朗读在后台运行，您可以同时浏览其他网页或应用。
