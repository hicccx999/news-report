### **综述**
本周 AI 领域迎来密集发布潮。OpenAI 正式推出了 **GPT-5.4** 系列，显著提升了计算机自动化操作（Computer Use）能力；Google 紧随其后更新了 **Gemini 3.1** 及其深度推理模式；开源界则由 **DeepSeek V4** 领衔，实现了万亿参数规模的国产模型开源突破。

---

### **核心新闻条目**

1. **OpenAI 发布 GPT-5.4 及其全系衍生模型**
   *   **动态：** OpenAI 于 3 月 18 日正式上线 GPT-5.4，同步推出 GPT-5.4 mini 和 nano。
   *   **突破：** 该模型引入了“超长上下文回忆”技术，并大幅优化了子智能体（Subagents）的编排效率。其 mini 版本在推理速度上比前代提升了 2 倍，专门针对自动化 UI 操作和实时代码生成进行了适配。

2. **Google Gemini 3.1 Pro 与“内置工具组合”功能上线**
   *   **动态：** Google 宣布 Gemini 3.1 Pro 全球推行，并为 Gemini API 引入了“内置工具与函数调用组合”功能。
   *   **突破：** 开发者现在可以在单次 API 调用中同时使用 Google Maps 等内置工具和自定义函数。此外，**Gemini 3 Deep Think** 模式通过模拟人类科研思维，在解决复杂工程问题上的准确率提升了 15%。

3. **DeepSeek V4 万亿参数开源模型震撼发布**
   *   **动态：** 中国 AI 领军企业 DeepSeek 发布了 V4 版本，采用改进的 MoE 架构，参数量级达到 1 万亿。
   *   **突破：** 该模型在多项基准测试中超越了 Claude 4.6，且训练成本仅为美国同类模型的 1/20。其开源权重已同步至 Hugging Face，成为目前全球最强的开源长文本模型之一。

4. **Yann LeCun 离开 Meta 创办 AMI Labs，获 10 亿美元种子轮融资**
   *   **动态：** 图灵奖得主 Yann LeCun 确认离开 Meta，创办旨在开发“世界模型（World Models）”的初创公司 AMI Labs。
   *   **意义：** 该实验室专注于非自回归式架构，目标是让 AI 像人类一样通过观察物理世界来学习，而非仅仅依赖文本训练。这一动作标志着 AI 基础架构竞争进入新赛道。

5. **Cerebras 与 AWS 达成合作，提供全球最快 AI 推理服务**
   *   **动态：** AWS 宣布在 Bedrock 平台上全面部署 Cerebras CS-3 系统。
   *   **技术：** 该架构将 AWS Trainium 芯片与 Cerebras 的晶圆级引擎结合，使 LLM 的每秒 Token 输出量（TPS）提升了 5 倍，大幅降低了大型智能体群（Agent Swarms）的运行成本。

6. **Anthropic 引入 Claude 4.6 “努力度控制（Effort Controls）”**
   *   **动态：** Anthropic 更新了 Claude 4.6 系列，新增了颗粒化的“努力度”调节选项。
   *   **功能：** 用户可根据任务复杂度在“低、中、高、极致”四个档位间切换，允许模型在处理法律合约审核或高级科研任务时，通过增加内部思考步骤（Chain-of-Thought）来消除幻觉。

7. **Nvidia “Vera Rubin” 平台进入量产测试阶段**
   *   **动态：** 英伟达宣布其下一代 AI 代工平台“Vera Rubin”及 H300 GPU 已开始向首批核心客户供货。
   *   **进展：** 该平台专门为万亿及十万亿参数模型设计，支持原生 FP8 稀疏解码，旨在支撑 2026 年底即将出现的超大规模实时模拟应用。
